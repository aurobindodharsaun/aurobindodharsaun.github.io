    HTML stands for HyperText Markup Language

    But what's that exactly?

    HTML is a computer text processing system, also known as a markup language, for annotating a document to be syntactically distinguishable from text, made specifically for creating web applications and describing their structure semantically.

    DOM - Document Object Model

    A platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.
